Write data from multiple buffers over a TCP socket.

The main difference with the library function is that this function handles EINTR (signal received while receiving data).

Parameters
sockfdSocket file descriptor
iovpointer vector points to a struct iovec defining address and bytes
iovcntNumber of blocks described into vector iov
Returns
  • On success, returns the number of bytes written
  • On error, -1 is returned, and errno is set appropriately
