This program is provided only as source form.
It is written in C (not C++).
It should compile w/o problem on any Linux Distribution and FreeBSD.

To download the source, use svn:
git clone git@github.com:osingla/messip.git

To compile the application, just follow these steps:
cd messip/messip-lib/Debug/
make
cd ../../messip-mgr/Debug/
make
cd ../../example-1/Debug
make
cd ../../messip-mgr/Debug/ ; ./messip_mgr

To run an example, just follow these steps:
  • On one shell, start the messip manager (do this only once)
    cd ../../messip-mgr/Debug/ ; ./messip_mgr

  • On a 2nd shell, start the first example (running as the server)
    cd ../../example-1/Debug/ ; LD_LIBRARY_PATH=../../messip-lib/Debug ./p server

  • On a 2nd shell, start the first example (running as the client)
    cd ../../example-1/Debug/ ; LD_LIBRARY_PATH=../../messip-lib/Debug ./p client
